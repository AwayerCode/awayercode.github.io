Welcome to my new blog! This is the first post on my Jekyll-powered blog hosted on GitHub Pages.

## Why Jekyll?

I chose Jekyll for several reasons:

1. **Simplicity** - Write in Markdown, get static HTML
2. **GitHub Pages Integration** - Free hosting with automatic deployment
3. **Performance** - Fast loading static pages
4. **Customization** - Full control over design and functionality

## What You'll Find Here

I plan to write about:

- **Technology** - Programming, web development, and software tools
- **Learning** - Notes and insights from my learning journey
- **Projects** - Updates on personal projects and experiments

## Features of This Blog

This blog includes several features I wanted:

- **Dark Mode** - Toggle between light and dark themes
- **Responsive Design** - Works on all devices
- **Search** - Find posts easily with client-side search
- **Tags & Categories** - Organize content by topic
- **Table of Contents** - Easy navigation within posts
- **Code Highlighting** - Beautiful syntax highlighting for code blocks

## Getting Started with Jekyll

If you're interested in starting your own Jekyll blog, here's what you need:

```bash
# Install Jekyll
gem install jekyll

# Create new site
jekyll new myblog

# Navigate to directory
cd myblog

# Start development server
bundle exec jekyll serve
```

Visit [Jekyll Documentation](https://jekyllrb.com/docs/) for more details.
